# Описание TechTask1

**TechTask1** - это проект, реализующий простой контроллер с таймером обратного отсчета, который отображает анимацию и визуальные элементы интерфейса для отсчета времени. 

<img width="288" alt="Screenshot 2024-11-04 at 3 07 15 PM" src="https://github.com/user-attachments/assets/25b4e47d-c0b5-459a-b6ee-f6cabb110ead">


### Особенности

- **Lottie-анимация**: используется анимация и воспроизводится в цикле.
- **SnapKit**: для контстрейнты элементов интерфейса.
- **Таймер**: оставшееся время обновляется каждую секунду, и таймер перезапускается после завершения отсчета.
